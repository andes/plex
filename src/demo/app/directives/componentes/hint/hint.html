La directiva hint se usa como una ayuda contextual avanzada.

<span hint="Un texto" hintType="<PlexType>" hintIcon="icon" detach="top | right | both"></span>
Hint   
  • Es multilínea
  • Se emplea sobre cualquier elemento, sea o no de acción
  • Opcionalmente se puede separar con detach arriba , detach derecha , detach ambos .
  • Si el elemento tiene una acción (click), el hint se vuelve clickeable
  • Sirve para indicar que hay alguna novedad o sugiere una funcionalidad
  • Puede darse un valor de importancia por color
  • Se puede usar a manera de "¿Sabías qué...?
  • Siempre visible como ícono flotante junto al elemento
  • Tiene un ícono que lo identifica (default: help)
  • El texto complementario se visualiza en el evento onMouseOver sobre el ícono
  • Según criterio, puede tener una función temporal, como indicar que hay una novedad
  • Para más opciones ver la documentación oficial de hint
