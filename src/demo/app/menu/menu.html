El componente plex-menu proporciona una interfaz de navegación consistente para la aplicación.

Propiedades principales:

  • menu: Array de ítems del menú
  • menuOpen: Estado de apertura del menú

Eventos:

  • menuToggled: Emitido cuando el menú se abre o cierra
  • collapseMenu: Emitido cuando se debe colapsar el menú
