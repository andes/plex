El componente plex-group-item se utiliza para incorporar un conjunto de elementos de forma ordenada.

  • Puede incluir plex-badge, plex-button y group-item para crear un selector de opciones.
  • Evita la necesidad de dar estilos a los componentes para ordenar y alinear.
  • Los elementos badge se ubican en primer lugar, los button en segundo y por último los item.

Puede incluirse cualquier tipo de botón o badge. Solo serán visibles si poseen atributo size="sm".
